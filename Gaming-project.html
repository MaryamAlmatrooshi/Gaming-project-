Cars : Python Game

The game created is called "Cars." It is one of the
most popular arcade games of all time. The player's
main goal in this game is to avoid crashing the car
and try to get a high score. There are 2 modes to the
game: one where there are four lanes moving in the same
direction, and the other mode where the lanes move in
opposite directions.
